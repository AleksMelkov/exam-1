Функция notEmpty()

    function noEmpty(arr) {
        if (!Array.isArray(arr)) {
            return 0;
        }
        return arr.filter((item) => {
            return item === '';
        }).length;
    }
Для имитации пустой строки в массиве поставить две и более запятых подряд.
Если оставить запятую на конце, это будет считаться за еще одну пустую строку

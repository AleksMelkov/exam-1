Главная

В левом окне пять ссылок на страницы, содержащие решение приведенных ниже задач. В правом окне картинка-ссылка, при нажатии на которую Вы попадаете на первую задачу. Каждая страница-задача содержит две картинки-ссылки – вперед и назад, позволяя перемещаться по задачам по порядку.

Задачи:

1. Создайте форму, состоящую из двух групп: «Личные данные» и «Начисления». В первую группу входят три текстовых поля: Фамилия, Имя, Отчество. Вторая группа содержит поля «Начислено», «Удержано» и «Итого», а также кнопку «Подсчитать».
В первой группе организовать доступ к полям по нажатию клавиш Alt+9, Alt+8, Alt+7. Необходима всплывающая подсказка, на какие клавиши нажимать.
Во второй группе после ввода в первые два поля чисел и нажатия на кнопку, в третьем поле должен отобразиться итог: Начислено-Удержано. Для перевода из текстового типа в числовой воспользуйтесь функцией parseInt.

2. На экран вывести три пары элементов Radio и Text. Каждому элементу Radio соответствует элемент Text. При выборе элемента Radio соответствующее текстовое поле должно активироваться. Одновременно можно выбрать только один элемент Radio. Когда Radio теряет фокус, текстовое поле деактивируется.

3. Разместите на странице элементы checkbox и select. Добейтесь, чтобы при щелчке на checkbox (галочка установлена) список select стал активным. При снятии галочки select должен деактивироваться. При этом выбор любого пункта списка приведет к выдаче сообщения о номере этого пункта и его значении (методом alert).

4. По нажатию кнопки текст, введенный в текстовое поле, переносится как надпись на кнопку.

5. Ввести форму с несколькими текстовыми полями. Используя методы focus() и select() добиться, чтобы по нажатию на кнопку фокус устанавливался в первое текстовое поле и его содержимое выделялось.
